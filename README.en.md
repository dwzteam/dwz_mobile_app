## dwz_mobile_app

> ### Description

- DWZ Mobile HTML5 + Hybrid（iOS, Android）

**Require:**

- node\^10.13.0
- npm\^6.14.5
- apicloud-cli\^0.2.0

> ### Quickstart

1. Download code for DWZ Mobile
```
git clone https://gitee.com/dwzteam/dwz_mobile_app.git
```
2. run for test

    - `npm intall`
    - `npm run dev`

3. [DWZ Mobile Doc](http://dwzteam.gitee.io/dwz_mobile_doc_v1)
